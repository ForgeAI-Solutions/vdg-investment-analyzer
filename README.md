# VDG Investment Analyzer

AI-powered investment analysis application for evaluating real estate opportunities
using configurable financial models and automated insights.

## Overview
This project supports long-term and short-term rental analysis by streamlining
property evaluation workflows, reducing manual calculations, and improving
decision consistency.

## Key Capabilities
- Long-term vs short-term rental comparison
- Cash flow and ROI calculations
- Configurable assumptions and inputs
- Automated financial insights

## Tech Stack
- TypeScript
- React
- Node.js
- Replit (deployment)
